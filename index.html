AI-BUILT PROJECT

March Madness Pool

A full-stack NCAA tournament pick-em app — designed, built, and shipped using AI as my development partner

🎯 The Problem

I wanted to run a friends-and-family NCAA tournament pool for 100+ participants with custom scoring rules — daily pick-em with upset bonuses, Elite Eight predictions, and a real-time leaderboard. No existing platform supported the format I needed.

Rather than compromise on the rules, I decided to build the platform myself — with no prior Python or web development experience — using AI tools as my development partner.

🛠 What I Built

  • ▶ User authentication — email/password sign-up, login, and password reset with Supabase Auth
  • ▶ Daily pick submission — users pick winners for each game before tip-off, with automatic lockout once games start
  • ▶ Elite Eight predictions — pre-tournament picks for which 8 teams will make the Elite Eight, earning bonus points
  • ▶ Automated scoring — a cron job pulls results from ESPN's API every 30 minutes and scores all picks, including a custom upset bonus point system
  • ▶ Smart defaults — auto-assigns the higher seed for any missed picks so no one falls behind
  • ▶ Live leaderboard — real-time standings with round-by-round point breakdowns and search
  • ▶ Multi-entry support — users can register multiple entries under one account, each scored independently

📸 The App

Sign Up — streamlined registration with entry name creation built into the sign-up flow
Pick Submission — daily matchups with seed info and potential point values, including upset bonus previews
Live Leaderboard — real-time standings with round-by-round scoring breakdown and Elite Eight bonus tracking

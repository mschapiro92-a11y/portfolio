ENTERPRISE AI ADOPTION

Custom Copilot Studio Agent for Large-Team Collaboration

How I used Microsoft Copilot Studio to solve a consistency problem across 20+ contributors and drive real AI adoption

🎯 The Problem

Large cross-functional teams produce high-stakes deliverables collaboratively — often with 20+ authors contributing to a single document. The result is inconsistent tone, conflicting messaging, and a painful review cycle where leadership has to manually realign the output every time.

The team needed a way to maintain one voice and stay aligned to strategic themes — without slowing anyone down or requiring every contributor to sit through a style briefing.

🛠 The Solution

I built a custom agent in Microsoft Copilot Studio that served as the team's always-available writing partner — grounded in domain-specific knowledge and configured with structured instructions that encoded quality standards.

  • ▶ Knowledge Sources — uploaded curated reference documents into Copilot Studio's knowledge base, giving the agent access to proven examples, approved language, and historical context. The agent could retrieve and reference this material when generating or refining content.
  • ▶ Instructions & System Prompt — used the agent's instruction notes to define tone, structure, and strategic themes. Incorporated leadership feedback and quality standards directly into the prompt so every response reflected the team's shared direction — not just generic AI output.
  • ▶ Topics & Conversation Design — configured specific topics within Copilot Studio to handle different content types (executive summaries, technical sections, strategy narratives), each with tailored trigger phrases and response patterns.
  • ▶ Iterative Refinement — continuously updated the knowledge base and instructions based on feedback loops with leadership, keeping the agent aligned as priorities evolved.

WORKFLOW AUTOMATION

Automated Executive Briefings with Power Automate & Copilot

How I built an automated pipeline that pulled from multiple data sources and delivered consistent, AI-enhanced briefing decks before every account meeting

🎯 The Problem

A senior stakeholder needed up-to-date briefing decks before every major account meeting. Each briefing required pulling from multiple disconnected sources — public financial data, recent press coverage, internal relationship history, and pipeline activity.

Preparing these manually meant hours of research, copy-pasting between systems, and inconsistent formatting. The information was always available — it just wasn't assembled. And every briefing had to look and feel the same, regardless of who prepared it or which account it was for.

🛠 The Solution

I designed a human-in-the-loop workflow that paired a manual Copilot research step with a fully automated Power Automate pipeline. Copilot handled the research — the flow handled everything else.

  • ▶ Copilot Research (Manual Step) — ran a saved M365 Copilot prompt to research the latest earnings reports, press releases, and publicly available company information for each account. Copilot output was reviewed and saved to a standardized Excel template with consistent columns and cell references.
  • ▶ Structured Excel Input — the Excel workbook used structured tables with fixed column headers (e.g., Company Overview, Key Financials, Recent News) so Power Automate could reliably read each cell by name — no parsing, no ambiguity.
  • ▶ SharePoint Data via OData Filters — the flow used SharePoint's "Get items" action with OData $filter queries (e.g., Account eq 'Contoso' and Date ge '2024-01-01') to pull engagement history and relationship data scoped to the specific account and time range.
  • ▶ Pipeline Integration — pulled current pipeline and opportunity data to give the stakeholder a real-time view of what was in motion with each account heading into the meeting.
  • ▶ Templated PowerPoint Output — Power Automate populated a PowerPoint template using the Populate a Word Template connector and Office Scripts, mapping Excel cells and SharePoint fields to specific slide placeholders for a consistent, executive-ready deck every time.

💡 The Takeaway

Not everything needs to be fully automated to be effective. This workflow succeeds because it puts human judgment where it matters — the research — and automates everything after. Copilot handles the open-ended intelligence gathering that requires a human eye, then a structured Excel handoff makes the rest of the pipeline reliable and repeatable. The best enterprise automations are hybrids: human-in-the-loop where quality matters, lights-out where consistency matters.

AI-BUILT PROJECT

March Madness Pool

A full-stack NCAA tournament pick-em app — designed, built, and shipped using AI as my development partner

🎯 The Problem

I wanted to run a friends-and-family NCAA tournament pool for 100+ participants with custom scoring rules — daily pick-em with upset bonuses, Elite Eight predictions, and a real-time leaderboard. No existing platform supported the format I needed.

Rather than compromise on the rules, I decided to build the platform myself — with no prior Python or web development experience — using AI tools as my development partner.

🛠 What I Built

  • ▶ User authentication — email/password sign-up, login, and password reset with Supabase Auth
  • ▶ Daily pick submission — users pick winners for each game before tip-off, with automatic lockout once games start
  • ▶ Elite Eight predictions — pre-tournament picks for which 8 teams will make the Elite Eight, earning bonus points
  • ▶ Automated scoring — a cron job pulls results from ESPN's API every 30 minutes and scores all picks, including a custom upset bonus point system
  • ▶ Smart defaults — auto-assigns the higher seed for any missed picks so no one falls behind
  • ▶ Live leaderboard — real-time standings with round-by-round point breakdowns and search
  • ▶ Multi-entry support — users can register multiple entries under one account, each scored independently

📸 The App

Sign Up — streamlined registration with entry name creation built into the sign-up flow
Pick Submission — daily matchups with seed info and potential point values, including upset bonus previews
Live Leaderboard — real-time standings with round-by-round scoring breakdown and Elite Eight bonus tracking
